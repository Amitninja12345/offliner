### Overview

Offliner currently supports three manifest-file formats:

* [Maven POM](format-maven-pom.html) (`pom.xml` or `*.pom`)
* Indy Folo Tracking Report JSON (`*.json`)
* Plaintext or Comma-Separated Values (any other file)
